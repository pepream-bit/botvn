# 🤖 DARK VN BOT

บอท Telegram สำหรับจัดการสมาชิกในกลุ่ม (แบน/ปลดแบน) พัฒนาด้วย Node.js และรันบนระบบ Cloud ของ Render เพื่อความเสถียรตลอด 24 ชั่วโมง

## 🛠️ ความสามารถของบอท
- **Ban Member:** สั่งแบนสมาชิกผ่าน ID พร้อมเตะออกจากกลุ่มทันที
- **Unban Member:** สั่งปลดแบนสมาชิก
- **Auto-Delete Notification:** ข้อความประกาศแบน/ปลดแบนในกลุ่มจะทำลายตัวเองอัตโนมัติภายใน 1 นาที เพื่อความเป็นระเบียบเรียบร้อยของกลุ่ม
- **Private Report:** บอทจะรายงานผลการทำงานสรุปสั้นๆ เข้าแชทส่วนตัวของคุณทันที

## 📋 คำสั่งใช้งาน (ในแชทส่วนตัวกับบอท)
พิมพ์คำสั่งผ่านแชทส่วนตัวกับบอทเพื่อใช้งาน:
- `/start` - ตรวจสอบสถานะการทำงานของบอท
- `/ban [ไอดีสมาชิก] [เหตุผล]` - สั่งแบนสมาชิกและลบประกาศใน 1 นาที
- `/unban [ไอดีสมาชิก] [เหตุผล]` - สั่งปลดแบนสมาชิก
